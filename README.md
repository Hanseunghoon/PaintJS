# [PaintJS](https://hanseunghoon.github.io/PaintJS/)

VanillaJS로 만든 페인팅 App

그리고 싶은 대로 그리고 저장할 수 있습니다.

## 기술

- HTML
- CSS
- Javascript
- API : [CanvasRenderingContext2D](https://developer.mozilla.org/en-US/docs/Web/API/CanvasRenderingContext2D)

## 특징

- 그리기 모드
- 브러쉬 굵기 조절
- 색 변경
- 채우기 기능
- 그림 저장
